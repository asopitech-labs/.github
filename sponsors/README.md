# GitHub Sponsors プロフィール文言の正本

https://github.com/sponsors/asopitech に表示される文言の正本（source of truth）をここで管理します。
GitHub Sponsors のプロフィールはダッシュボードでしか編集できないため、運用は次のルールで行います。

## 運用ルール

1. 文言の変更は **必ずこのリポジトリの Markdown を先に編集**し、コミットする（履歴 = 変更履歴）。
2. コミット後、[Sponsors ダッシュボード](https://github.com/sponsors/asopitech/dashboard/profile) に内容を貼り付けて反映する。
3. 反映したら、コミットメッセージまたは下記の反映記録に日付を残す。

## ファイル構成

| ファイル | 対応する画面 |
| --- | --- |
| [`PROFILE.md`](./PROFILE.md) | プロフィールの紹介文（Introduction） |
| [`TIERS.md`](./TIERS.md) | スポンサーティアの説明文 |
